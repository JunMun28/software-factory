### 4. After a refresh on the review page, 'Edit' silently creates a duplicate request and orphans the interview answers
- **Dimension:** code · **Where:** `apps/intake/src/app/submitter/review.ts:28`
- **Evidence:** Review never restores the draft: its constructor (lines 160-162) only fetches the request; unlike Interview it does not call `draft.loadAttachments(this.id)`, so after a full page reload `draft.requestId` is null.

# Image-Generation Prompts — one per screen

> **Brand: Micron.** Black/white base with a single vivid **Micron purple `#BD03F7`** accent used minimally and intentionally (primary actions, focus, active/selected, brand moments). Amber and red are **functional only** (a gate needs you / Needs-human) — not brand. Purple **gradients appear only at brand moments** (login hero, empty states, logo lockup); all working surfaces (board, feed, forms, detail) stay flat for legibility.
>
> Paste a prompt into your image model to visualize the screen. Each prompt is self-contained and carries this **shared visual style** (light theme by default across both the Control center and the Submitter side — dark is an opt-in; Inter, hairline borders, dense Linear-like layout) so all screens read as one product. Render at 16:10, desktop, high fidelity unless the screen notes mobile.

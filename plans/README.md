## Findings considered and rejected (do not re-audit)

From the 2026-06-11 audit (4 parallel auditors, every finding vetted against
the code before judgment):

- **Approve double-fire race** (`main.py` approve): already handled — the
  atomic `UPDATE … WHERE status='pending_approval'` claim means only the
  winner emits events; the loser takes the idempotent path. A test asserts
  single emission (`test_approve_ledger_and_idempotent_replay`).
- **"Subscription leak" in components** (`api.apps().subscribe` etc.):
  HttpClient observables complete after one emission; not a leak.
- **Stale `pending_question` after send-back**: the interview only runs in
  draft/submitted; after submission no UI or endpoint path re-reads it.
- **Docker `--host 0.0.0.0`**: required for nginx→api proxying inside the
  compose network; the api port is never published to the host.
- **N+1 / lazy-load claims on detail + comments endpoints**: each is a
  single bounded query (or ~4 lazy loads per detail fetch) — fine at this
  scale; `selectinload` is an optional micro-optimization, not a finding.
- **Duplicate-detection scan cost**: already status-gated to pre-approval
  and bounded to 200 rows (the code comment documents the earlier fix).
- **Simulator per-tick scan**: single indexed-ish query over a small table;
  ADR 0013 already addressed the real polling costs.
- **Ledger replay test gap**: the test already asserts `repo_ready`,
  `spec_pr_open`, `stage2_fired` and event-count stability on replay.
- **Claude-runner gate tests assert only escalation**: false — they assert
  the escalation REASON ("Test-isolation gate", "RED gate") and the rollback
  of the weakened test file.
- **Smoke.sh missing failure scenarios**: covered in pytest instead
  (`test_architecture.py`: orphan rescan, crashed stage, merge failure,
  cancel-beats-merge).
- **Subprocess env allow-list** (claude_exec): deferred deliberately — the
  CLI needs ambient auth; an allow-list buys little in a local tool and
  risks breaking `FACTORY_RUNNER=claude`. Revisit if deployed beyond
  localhost (noted in plan 005's out-of-scope).
